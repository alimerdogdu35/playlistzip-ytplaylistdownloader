# StreamZip 🚀 - YouTube Playlist to MP3 Downloader

YouTube playlistlerini paralel indirme motoru kullanarak saniyeler içinde toplu olarak MP3 formatına dönüştüren profesyonel bir web uygulaması.

## 🌟 Özellikler
- **Hız:** Paralel indirme motoru (5x daha hızlı).
- **Limit Yok:** 50+ video içeren dev playlist desteği.
- **Kalite:** 320kbps yüksek ses kalitesi.
- **SEO Uyumlu:** Modern ve kullanıcı dostu arayüz.

## 🛠 Kurulum
1. Projeyi klonlayın: `git clone https://github.com/kullaniciadi/streamzip.git`
2. Bağımlılıkları kurun: `npm install`
3. Sisteminizde `ffmpeg` ve `yt-dlp` yüklü olduğundan emin olun.
4. `.env.example` dosyasını `.env` olarak değiştirin ve API Key'inizi girin.
5. Başlatın: `npm start`
